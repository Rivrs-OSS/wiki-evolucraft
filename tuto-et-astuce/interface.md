# 🖼️ Problème d'interface

## <mark style="color:green;">💠 Comment voir ses compétences et les barres de boss ? 🖼️</mark>

Vous avez bien suivi les instructions pour **<mark style="color:green;"><strong>installer les compétences</strong></mark>** et vous ne voyez toujours rien ? De même pour les **<mark style="color:green;"><strong>barres de boss</strong></mark>** dans les donjons ? Pas d'inquiétude, il existe possiblement une solution !

### <mark style="color:green;">🔸 Étape 1️⃣</mark>
Appuyez sur la touche **<mark style="color:green;"><strong>"Échap"</strong></mark>** de votre clavier afin d’ouvrir le **<mark style="color:green;"><strong>menu de pause</strong></mark>**.

### <mark style="color:green;">🔸 Étape 2️⃣</mark>
Cliquez sur le bouton **<mark style="color:green;"><strong>"Options"</strong></mark>** afin d’accéder aux **<mark style="color:green;"><strong>paramètres du jeu</strong></mark>**, comme indiqué sur l’image ci-dessous.
<figure><img src="../.gitbook/assets/Tuto_Et_Astuce/Interface/Menu.png" alt="" width="600"></figure>

### <mark style="color:green;">🔸 Étape 3️⃣</mark>
Cliquez sur le bouton **<mark style="color:green;"><strong>"Graphismes"</strong></mark>** comme indiqué ci-dessous afin de paramétrer tout ce qui concerne le **<mark style="color:green;"><strong>graphisme du jeu</strong></mark>**.
<figure><img src="../.gitbook/assets/Tuto_Et_Astuce/Interface/Options.png" alt="" width="600"></figure>

### <mark style="color:green;">🔸 Étape 4️⃣</mark>
Cliquez sur le bouton **<mark style="color:green;"><strong>"Taille de l'interface"</strong></mark>** jusqu’à ce que l’interface soit réglée sur **<mark style="color:green;"><strong>taille 2</strong></mark>**, comme sur l’image ci-dessous.  
Cela permettra de **<mark style="color:green;"><strong>réduire toutes les interfaces</strong></mark>** (y compris votre **<mark style="color:green;"><strong>inventaire</strong></mark>** et votre **<mark style="color:green;"><strong>barre d’action</strong></mark>**).
<figure><img src="../.gitbook/assets/Tuto_Et_Astuce/Interface/Graphisme.png" alt="" width="600"></figure>

{% hint style="warning" %}
Il est possible que si vous jouez avec le mod **<mark style="color:green;"><strong>Sodium</strong></mark>**, l’interface ne soit pas exactement la même que ci-dessus.  
Dans ce cas, suivez les étapes jusqu’à **<mark style="color:green;"><strong>l’étape 3</strong></mark>** afin d’arriver sur l’interface ci-dessous, puis déplacez le **<mark style="color:green;"><strong>curseur de la taille de l’interface</strong></mark>** pour le régler sur **<mark style="color:green;"><strong>2</strong></mark>**.
<figure><img src="../.gitbook/assets/Tuto_Et_Astuce/Interface/Sodium.png" alt="" width="600"></figure>
{% endhint %}

Vous voilà désormais avec une **<mark style="color:green;"><strong>meilleure visibilité de vos compétences</strong></mark>** ! 🥳
